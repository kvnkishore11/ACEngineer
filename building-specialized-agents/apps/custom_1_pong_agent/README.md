# Pong Agent - Level 0.1

The simplest possible agent that responds with 'pong' to any input.

## Python Setup

```bash
# Install dependencies
uv sync

# Run the agent
uv run python pong_agent.py
```

## TypeScript Setup

```bash
# Install dependencies
bun install

# Run the agent
bun run pong_agent.ts
```

## What It Does

- Takes any input (we send "ping")
- Always responds with exactly "pong"
- Demonstrates the most basic agent interaction with Claude Agent SDK
