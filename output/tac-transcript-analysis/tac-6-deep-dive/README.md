## Core Innovation: One Agent, One Prompt, One Purpose

The central tactic of TAC-6 challenges conventional thinking about AI agents:

> "Massive context windows often leads to a distracted, confused agent. This is context pollution. This is context overloading. This is toxic context."

Instead, TAC-6 advocates for:
- **Specialized agents** with singular focus
- **Minimal context** for maximum performance
- **Improvable prompts** in version control
- **Evaluatable workflows** for continuous improvement

## The Five Questions of Engineering

TAC-6 completes the Software Developer Lifecycle by framing each step as a question:
