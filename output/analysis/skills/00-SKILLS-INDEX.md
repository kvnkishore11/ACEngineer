## ðŸ“‹ Complete Skills Catalog

### 1. **Agentic Architect** (`agentic-architect.md`)
**Purpose**: Design complete agentic system architectures
**When to Use**: Starting new projects, refactoring existing systems, choosing architectural patterns
**Key Capabilities**:
- Select from 5 architectural patterns (TAC-8)
- Design ADW (Agentic Development Workflow) systems
- Create multi-agent orchestration architectures
- Plan system integration strategies

### 2. **Agent Builder** (`agent-builder.md`)
**Purpose**: Create specialized agents for specific tasks
**When to Use**: Need custom agent behavior, building domain-specific agents
**Key Capabilities**:
- Build agents from basic to advanced (8 types from Horizon)
- Configure agent tools and permissions
- Design agent communication protocols
- Implement stateful agent behaviors

### 3. **Prompt Engineer** (`prompt-engineer.md`)
**Purpose**: Master all 7 levels of prompt engineering
**When to Use**: Creating new prompts, optimizing existing prompts, building meta-prompts
**Key Capabilities**:
- Progress through 7 prompt levels
- Create self-improving prompts
- Build prompt templates and generators
- Design delegation prompts

### 4. **Context Optimizer** (`context-optimizer.md`)
**Purpose**: Apply R&D framework for maximum agent performance
**When to Use**: Agent hitting context limits, slow performance, need to scale
**Key Capabilities**:
- Measure context usage
- Apply reduction techniques (MCP, CLAUDE.md, Output control)
- Implement delegation patterns
- Create context bundles

### 5. **Workflow Designer** (`workflow-designer.md`)
**Purpose**: Design complete SDLC workflows and pipelines
**When to Use**: Setting up development processes, automating workflows
**Key Capabilities**:
- Create BFC (Bug/Feature/Chore) workflows
- Design ISO (Issue-Solution-Output) patterns
- Build ADW pipelines
- Implement ZTE (Zero Touch Execution)

### 6. **Integration Specialist** (`integration-specialist.md`)
**Purpose**: Integrate agentic systems with existing infrastructure
**When to Use**: Production deployment, connecting to databases/APIs, MCP integration
**Key Capabilities**:
- MCP server integration
- API and WebSocket implementation
- Database connectivity (PostgreSQL patterns)
- CI/CD pipeline integration

### 7. **Testing Strategist** (`testing-strategist.md`)
**Purpose**: Test and validate agentic systems
**When to Use**: Quality assurance, debugging agents, validation workflows
**Key Capabilities**:
- Test agent behaviors
- Validate workflows end-to-end
- Create test automation (Playwright patterns)
- Debug multi-agent interactions

### 8. **Agentic Instructor** (`agentic-instructor.md`)
**Purpose**: Teach agentic concepts and create documentation
**When to Use**: Onboarding team members, documenting systems, creating tutorials
**Key Capabilities**:
- Explain agentic patterns
- Create learning paths (TAC-1 to TAC-8 progression)
- Document system architectures
- Build interactive tutorials
